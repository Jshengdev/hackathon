# Step 6: Yap Mode

## STEP GOAL:

Capture the unstructured insights that often contain gold.

## MANDATORY SEQUENCE

### 1. Activate Yap Mode

"**Step 5: Yap Mode Activated 🎤**

This is your chance to dump ANYTHING you've noticed or overheard:
- Random insights from sponsor booths
- Overheard judge conversations
- Patterns from past winners (if posted)
- Gut feelings about tracks
- Wild ideas (even if incomplete)
- Technical curiosities
- Anything that might matter

**No filter. No organization. Just yap.**

I'll extract the signal.

What have you noticed?"

### 2. Accept All Input

**MAXIMUM YAP MODE**

Accept:
- Stream of consciousness
- Random observations
- Half-formed ideas
- Technical curiosities
- Gut feelings
- Contradictions
- Tangents

**No judgment. Just capture.**

### 3. Extract and Categorize

As they yap, identify:
- **Actionable insights** — Something we can use
- **Pattern observations** — About judges, winners, tracks
- **Technical curiosities** — Could become idea seeds
- **Gut feelings** — Worth noting even if unsubstantiated

### 4. Write Yap Files

For each yap session/topic, write: `{outputLocation}/context/yaps/{timestamp}.md`

```markdown
# Yap Session - [timestamp]

## Raw Input
[Exactly what they said, unfiltered]

## Extracted Insights
- [Actionable insight 1]
- [Actionable insight 2]

## Pattern Observations
- [Pattern about judges/tracks/winners]

## Technical Curiosities
- [Interesting tech that came up]

## Gut Feelings
- [Unsubstantiated but worth noting]

## Potential Idea Seeds
[Anything that could become a project idea]
```

### 5. Confirm Collection Complete

"**✓ Yaps captured**

I've collected everything. Now I'll consolidate all this context into one clean file with track-stacking analysis."

### 6. Auto-Proceed

Load, read entire file, then execute `{nextStepFile}`.

---

**Master Rule:** Signal often hides in noise. Capture everything, filter later.

## 🚨 SYSTEM SUCCESS/FAILURE METRICS

### ✅ SUCCESS:

- Full yap mode activated (no filtering)
- All insights captured in yap files
- Signal extracted from noise
- Actionable insights identified
- Ready for consolidation

### ❌ SYSTEM FAILURE:

- Filtering or judging during yap mode
- Missing random insights that could be gold
- Not capturing gut feelings
- Skipping yap extraction

**Master Rule:** Signal often hides in noise. Capture everything, filter later.
