## Strategic Read
- These are **multi-stack-easy** — most can be appended to a main project with minimal extra work
- Highest-value adds for our thesis:
  - **Gemini API** — if we're using a multimodal/vision model anyway, swap to Gemini and tag it
  - **Mem0** — if we're tracking what users consume over time, persistent memory IS the architecture
  - **Auth0** — only if we have user accounts; might be free win
  - **GoDaddy domain** — pick a sticky/clever .com
- AVOID forcing Vultr/Solana/etc. unless they're already in our stack — these are easy to over-stack and dilute our pitch
