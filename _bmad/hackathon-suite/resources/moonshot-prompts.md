# Moonshot Prompts

Curated prompts for hackathon ideation. Use these to push past obvious ideas into "probe the dark" territory.

---

## Core Moonshot Prompts

### The Impossible Question
> "What's the most IMPOSSIBLE idea for [Track X]?"

Force yourself past feasibility. The first impossible idea often contains the seed of the winning idea.

### The Resource Explosion
> "If you had 1000 engineers for 6 hours, what would you build?"

Remove skill constraints. What would the ideal solution look like?

### The Judge Reaction
> "What would make a judge say 'holy shit' out loud?"

Optimize for memorable, not achievable. Then work backwards.

### The Time Probe
> "What's 10 years from now, built today as a demo?"

Future vision compressed into present proof-of-concept.

### The Dark Probe
> "Look into the future and probe into the dark. What's there?"

Let imagination lead. Structure follows.

## Track-Specific Prompts

### Hardware Track
> "What physical interaction would blow someone's mind?"
> "What sensor data has never been combined before?"
> "What would make someone want to touch your demo?"

### AI/ML Track
> "What would AI do that seems impossible but might work once?"
> "What's the weirdest input → output transformation?"
> "What would make Gemini/GPT feel like magic?"

### Social Impact Track
> "What problem do we pretend doesn't exist?"
> "Who gets ignored by current technology?"
> "What would change a life if it worked?"

### Sustainability Track
> "What's the most wasteful thing we do that tech could fix?"
> "What would net-zero look like for [industry]?"
> "What if energy was the constraint we designed around?"

### Gaming/Entertainment Track
> "What game mechanic doesn't exist but should?"
> "What would make someone laugh AND be impressed?"
> "What play experience is impossible... until now?"

---

## Combination Prompts

### Cross-Pollination
> "What happens when [Hardware] meets [AI] meets [Social Impact]?"
> "What's the intersection of [Track A] and [Track B] that no one's thought of?"
> "Steal the best idea from [Track X] and apply it to [Track Y]."

### Constraint Fusion
> "Build for someone who can't [see/hear/move/speak]. What emerges?"
> "What if the network was 100x slower? What becomes valuable?"
> "What if there was no cloud—only edge?"

### Pattern Breaking
> "What assumption does every other team make that we could violate?"
> "What's the opposite of the obvious solution?"
> "What would a 5-year-old build with these tools?"

## Weird Energy Prompts

### The Random Combination
> "Accelerometers + body language + [random API] = ?"
> "What if neural network weights were physical resistors?"
> "What if the output was a smell instead of a screen?"

### The Scale Thought Experiment
> "What if this worked at 1000x scale? What would matter?"
> "What if only ONE person could use this—who should it be?"
> "What if this had to work for 100 years?"

### The Absurdity Gateway
> "What's the dumbest version of this that might actually work?"
> "What would aliens find weird about how we solve this?"
> "What would this look like if it was designed by a cat?"

## Prompts for Getting Unstuck

### When Ideas Feel Boring
> "Why would anyone remember this tomorrow?"
> "What would make this illegal? (then make it legal but keep the energy)"
> "What's the version of this that would scare investors... in a good way?"

### When Ideas Feel Impossible
> "What's the minimum demo that proves this could exist?"
> "What if we only had to make it work ONCE?"
> "What's the one screenshot that would go viral?"

### When Ideas Feel Too Safe
> "What's the version that could fail spectacularly?"
> "What would the 10x version look like?"
> "What would make this team say 'holy shit we're actually doing this'?"

## Using These Prompts

### The Moonshot → Practical Flow

1. **Generate** — Use moonshot prompts, no filter
2. **Capture** — Write down everything, even ridiculous ideas
3. **Pattern** — Look for recurring themes across wild ideas
4. **Extract** — Find the kernel of "wow" in each
5. **Ground** — Ask "what's the one demo case that proves this?"
6. **Validate** — Run through Gap filter

### Prompt Rotation

If stuck, switch prompt categories:
- Technical prompts → Social prompts
- Scale prompts → Constraint prompts
- Serious prompts → Absurd prompts

Fresh angles unlock fresh ideas.

### Team Prompt Jam

Everyone answers the same prompt simultaneously (2 min silent). Then share. Build on each other's weirdest ideas.

---

## The Meta-Prompt

When all else fails:

> "What would win this hackathon that NO ONE else will think of?"

Then sit in the discomfort until something emerges.
