# Step 3: Present Sync Summary

## STEP GOAL:

Present a clear, actionable sync summary that gets the user re-oriented in < 30 seconds.

## MANDATORY EXECUTION RULES (READ FIRST):

### Universal Rules:

- 📖 CRITICAL: Read the complete step file before taking any action
- 🎯 ALWAYS follow the exact instructions in the step file
- This is the FINAL step - no next step

### Role Reinforcement:

- You are delivering a trusted assistant briefing
- Be clear and actionable
- User should feel oriented after reading

### Step-Specific Rules:

- 🎯 Focus on clear, scannable presentation
- 🚫 FORBIDDEN to make state changes
- 💬 This is the payoff - make it count
- ⏱️ Aim for < 30 second read time

## MANDATORY SEQUENCE

**CRITICAL:** Follow this sequence exactly. Do not skip, reorder, or improvise.

### 1. Present Sync Summary

Display using this format:

```
============================================================
                      SYNC SUMMARY
============================================================

LAST SESSION
------------
[Session name/description]
[When: date/time]
[What you were doing]

SINCE THEN
----------
[Changes - decisions made, sessions completed, etc.]
[Or "No changes" if nothing changed]

CURRENT STATE
-------------
Focus: [current focus]
Completed: [count or brief list]
In Progress: [what's active]

BLOCKERS
--------
[List blockers or "None - clear to proceed"]

------------------------------------------------------------
                    SUGGESTED NEXT
------------------------------------------------------------
[Clear recommendation for what to do now]

============================================================
```

### 2. Offer Quick Actions

"**Ready to:**
- [Continue where you left off]
- [Address a blocker if any]
- [Start a new session]

What would you like to do?"

### 3. Complete Workflow

User is now synced and can proceed with their chosen action.

"**You're synced.** Let me know what you'd like to work on."

---

## SYSTEM SUCCESS/FAILURE METRICS:

### SUCCESS:

- Summary presented clearly
- User can read in < 30 seconds
- Actionable suggestion provided
- User feels re-oriented

### FAILURE:

- Information overload (too much detail)
- Missing key information
- No clear next action
- User still confused after reading

**Master Rule:** Like a trusted assistant briefing - clear, concise, actionable.
