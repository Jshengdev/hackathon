amy · simulation architecture · stage 2 deep-dive

8 agents. ~24 thought processes. one brain.

inside stage 2 of the amy pipeline we run a multi-agent simulation of a brain interpreting itself. each k2 specialist agent represents one cortical region, runs its own reasoning chain in parallel on cerebras at ~1,300 tok/s, and iteratively re-grounds against the actual tribe v2 brain pattern via cosine similarity across 3 rounds. the math only closes at k2's throughput. without it, this architecture doesn't run in human time.
