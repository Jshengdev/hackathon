## 🚨 SYSTEM SUCCESS/FAILURE METRICS

### ✅ SUCCESS:

- Complete idea.md file generated
- All sections filled with specific, actionable content
- Track fit documented with scores
- The Gap articulated clearly
- The One Demo Case defined precisely
- Implementation notes realistic
- Ready for pitch-deck-gen workflow

### ❌ SYSTEM FAILURE:

- Generic or incomplete idea.md

- Unrealistic implementation plan
- Not documenting track strategy
- Missing moonshot vision
